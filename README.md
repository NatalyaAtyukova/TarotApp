Добро пожаловать в Tarot — приложение для тех, кто ищет ответы на важные вопросы и хочет прикоснуться к магии Таро!

С Tarot вы можете:
	•	Вытащить одну карту для быстрого гадания.
	•	Сделать расклад из трёх карт (прошлое, настоящее, будущее).
	•	Получить доступ к премиум-раскладам из пяти и десяти карт для глубокого анализа (доступно по подписке).
	•	Сохранить результат гадания, чтобы вернуться к нему позже.

Подписки:
	•	Базовая подписка открывает неограниченный доступ к раскладам на три карты.
	•	Премиум подписка позволяет использовать расклады на пять и десять карт, раскрывая все аспекты ситуации.

Простота, удобство и интуитивно понятный интерфейс сделают ваши сеансы Таро особенными. Исследуйте глубины своей интуиции, управляйте раскладами и находите ответы на важные вопросы с Tarot.
